# 📄 Research Paper Explainer

A simple AI-powered Research Paper Explainer built using **LangChain**, **Hugging Face**, **FAISS**, and **Streamlit**.

Upload any research paper in PDF format and get an easy-to-understand explanation including:

* 📝 Executive Summary
* 🎯 Problem Statement
* 🚀 Key Contributions
* ⚙️ Methodology
* 📊 Results & Findings
* ✅ Strengths
* ❌ Limitations
* 🔮 Future Work
* 👦 Explain Like I'm 15 (ELI15)

---

## 🏗️ Tech Stack

* LangChain
* Hugging Face Models
* FAISS Vector Database
* Sentence Transformers
* Streamlit
* PyPDF
